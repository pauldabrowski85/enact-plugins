# enact-plugins

> Claude Code plugins by [EnactSkill](https://enactskill.com).

## Install the marketplace

```
/plugin marketplace add https://github.com/pauldabrowski85/enact-plugins.git
```

## Plugins

### [motion-archaeologist](./motion-archaeologist)

Reverse-engineer how any website's animation is built — transport, timing,
easing, orchestration — and lift the keyframe / shape / asset data to disk.

```
/plugin install motion-archaeologist@enact-plugins
```
